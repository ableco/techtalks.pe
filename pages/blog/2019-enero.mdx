# Resumen de Tech Talks de enero 2019

Este <time>martes 15 de enero</time> tuvimos el primer Tech Talks del 2019, y el primero de su historia. Contamos con la presencia de más de 30 personas y dos charlas, Introducción a React.js con [Kattya Cuevas](https://twitter.com/KattyaCuevas) y un live-programming de una aplicación móvil usando Flutter con [Hansy Schmitt](https://twitter.com/hacksy_).

Desde la organización estamos muy agradecidos a todos los asistentes con el éxito que fue este primer Tech Talks y esperamos que hayan disfrutado tanto como nosotros organizarlo.

## Introducción a React.js

Kattya nos dió una charla donde nos contó un poco sobre su proceso de aprendizaje de React.js siendo desarrolladores Backend.

## Programando una aplicación con Flutter en vivo

Hansy se puso manos a la obra y nos guió en el proceso de crear una aplicación móvil usando Flutter, esta es una tecnología que nos permite usar Dart para crear aplicaciones multi plataforma para iOS y Android.
